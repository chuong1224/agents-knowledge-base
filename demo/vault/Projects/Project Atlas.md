# Project Atlas

Atlas is the working experiment behind this demo: AI agents do the reading and writing in a markdown vault, a human reviews and steers, and every agent action is logged and visualized.

## Principles
- The vault stays plain markdown — any tool can read it, nothing is locked in
- Agents log every file operation → activity can be replayed and measured
- Navigation hubs ([[Index - Research]], [[Index - Projects]]) keep retrieval cheap

Builds on [[Agent Memory Systems]] and [[Multi-Agent Orchestration]]; measured with [[Evaluation Harnesses]]. Status tracked in [[Weekly Review]].
