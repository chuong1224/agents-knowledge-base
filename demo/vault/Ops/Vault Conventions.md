# Vault Conventions

- Every note: `title`, `tags`, one-line `summary` in frontmatter
- Tags are a controlled vocabulary — research / skill / personal / vault-operation
- Link to the nearest index, let indexes link upward
- Notes link liberally: a link that doesn't resolve yet marks future work

Enforced by the checks in [[Vault Health Checks]].
