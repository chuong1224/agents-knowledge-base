# Python Testing Patterns

Fast suites get run; slow suites get skipped. Keep the default path under a few seconds and push integration tests behind a flag.

- Fixtures over setup methods
- Contract tests encode every bug you've already fixed
- One assert-story per test

Applied throughout [[Project Atlas]].
