# Linking Notes

Type a note's name inside double square brackets:

```
Read [[What Is a Note]] before this one.
```

That's a **wikilink**. It does three things at once:

1. In the Reader, it becomes clickable.
2. On the 3D graph, it becomes a **line between two nodes**. Every link you write reshapes the galaxy.
3. The target note gains a **backlink** — scroll to the bottom of any note in the Reader to see who points at it.

## Rules of thumb

- Link the note's **file name** (without `.md`): `[[My First Note]]`
- A link to a note that doesn't exist yet is fine — it marks something worth writing later
- Link generously. A note with zero links is an island; islands are where notes go to be forgotten

The graph you saw in the demo is nothing but a few hundred of these links. Now go add one: put a wikilink to this note inside [[My First Note]].
