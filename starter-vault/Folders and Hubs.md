# Folders and Hubs

## Folders

Folders in a vault are ordinary folders. Group notes however feels natural — by area (`Research/`, `Personal/`), by project, or not at all while you're under ~30 notes. The app shows the tree in the left sidebar either way.

## Hub notes

A **hub** (also called an *index* or *MOC — map of content*) is a note whose main job is linking to other notes:

```
# Index - Cooking
- [[Weeknight Pasta]]
- [[Sourdough Log]]
- [[Knife Care]]
```

Give hubs the `index` tag and they render as bright white nodes — you can see them holding the demo vault's continents together. Two habits make hubs work:

1. When you create a note, add a link to it from the nearest hub
2. Let hubs link downward and notes link sideways — avoid every note linking to everything

That's all the structure a few hundred notes need. [[Start Here]] is itself a hub — you're already using the pattern.
